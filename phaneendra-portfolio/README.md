## ✨ About This Portfolio

A modern, fully responsive **Neon Glassmorphism** portfolio website built with pure HTML, CSS, and JavaScript — no frameworks needed. Designed to showcase my work in Cybersecurity, AI/ML, and Full Stack Development.

---

## 🎨 Design Features

- 🔮 **Neon Glassmorphism** — frosted glass cards with cyan/purple glow borders
- ✨ **Particle Background** — animated floating particles with connection lines
- ⌨️ **Typing Effect** — rotating taglines with typewriter animation
- 📜 **Scroll Reveal** — sections animate in as you scroll
- 🖱️ **Custom Cursor** — glowing cursor that reacts to hover elements
- 📱 **Fully Responsive** — works on all screen sizes
- 🌙 **Dark Theme** — deep navy + purple + cyan color palette
